⚡ АВТОМАТИЧНИЙ APK через GitHub
Безкоштовно, без реклами, справжній APK! GitHub сам збере його на своїх серверах.
1
Зареєструйся на github.com → New repository → Public → Create
2
Натисни "uploading an existing file" → перетягни ВСІ файли з ZIP → Commit changes
3
Відкрий вкладку Actions → побачиш "Build APK" — запуститься автоматично!
4
Чекай 3-7 хвилин → натисни на workflow → Artifacts → скачай APK 🎉
5
Наступного разу — просто завантажуй новий index.html і APK збереться знову автоматично!
📱 AppsGeyser (альтернатива)
Завантаж index.html → Create App → Build APK
📱 PWA — Додаток без APK
1
Зареєструйся на github.com (безкоштовно)
2
Створи новий репозиторій → завантаж усі файли з ZIP
3
Settings → Pages → Source: main branch → Save
4
Відкрий посилання на телефоні в Chrome → три крапки → "Додати на головний екран"
5
Готово! Додаток на головному екрані, працює офлайн, без реклами!
